# Issue #1: READMEにCIバッジを追加

## 概要
READMEファイルにCI/CDパイプラインのステータスバッジを追加する

## 背景
現在のREADMEにはプロジェクトのビルド状態やテストカバレッジなどが視覚的に表示されていません。これらの情報をバッジとして表示することで、プロジェクトの品質状態を一目で確認できるようになります。

## 提案
以下のバッジをREADMEに追加:
- GitHub Actions CI/CDのビルドステータスバッジ
- テストカバレッジバッジ（必要に応じて）
- その他関連するステータスバッジ

## 期待される効果
- プロジェクトの品質状態の可視化
- 開発者やユーザーへの信頼性向上
- CI/CDパイプラインの稼働状況の即座な確認
